3:T1be2,<h1>1. Yahboom Jetson Nano Linux OS 설정</h1>
<p>중국산 젯슨나노, 즉 야붐 제품을 사용할 경우, 기존과 달리 USB내에 os를 설치하고 이어서 기기 내 EMMC에도 OS를 설치해줘야 한다.</p>
<p>이때 usb내의 OS와 emmc에 flash하는 os의 jetpack의 version을 동일하게 설정해줘야한다.</p>
<p>(참고 사이트 1번과 2번을 보면서 설치하면 된다.)</p>
<p><strong>그리고 리눅스로 sudo apt-get update를 진행하거나 (블라블라~)</strong></p>
<p><strong>아무튼 명령어(install, update, upgrade, etc) 입력해줄 때, 네트워크 설정과 날짜와 시간이 올바르게 설정되어 있는지 확인해줘야 한다!!! 이거 안되어 있으면 아무리 명령해도 진행 안됩니다.</strong></p>
<h2>1. emmc에 os 설치하기</h2>
<p>본 과정에서는 VM Ware 를 이용하여 Jetson Nano 내에 OS를 입력해줘야한다.</p>
<p>아래의 순서대로 진행하여 본다.</p>
<h3>vm ware안에서 NVIDIA SDK를 다운받아 준다.</h3>
<p><img src="https://user-images.githubusercontent.com/84653623/219853912-cb2d14e0-52c9-4fa0-989e-167c4c78892b.png" alt="10"></p>
<h3>Download로 이동하여 다운받은 파일 확인 및 설치 실행.</h3>
<p><img src="https://user-images.githubusercontent.com/84653623/219853914-cd1979b4-0081-4d63-8ade-63f7d0d30495.png" alt="11"></p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853917-b3e7f1bb-9cf2-4a8c-b165-09bead7be84f.png" alt="12"></p>
<p>다운 받는 과정에서 오류가 날 경우 다음과 같은 명령어를 입력하여 설치를 진행하여 준다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853911-7d1deaea-6981-4e9b-a5db-fa4dc41ab246.png" alt="9"></p>
<h3>SDK manager를 검색하여 실행. 이후 nvidia 계정으로 로그인 후 버전을 설정해주면 되는데, yahboom 공홈에 업로드 되어 있는 이미지의 jetpack 버전은 4.4.1이므로 이와 동일한 버전으로 flash해줘야한다.</h3>
<p><img src="https://user-images.githubusercontent.com/84653623/219853908-c80e1390-05ba-4d72-95b2-ffc3600bfed8.png" alt="5"></p>
<p>범퍼를 꽂아서 젯슨나노에 입력이 가능하도록 한다. 이후 vm ware가 있는 컴퓨터와 젯슨나노를 연결하여 준다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853922-77ffe7f2-c925-4121-9b00-c6c1e4c8e711.png" alt="19"></p>
<p>컴퓨터와 연결하는 선은 흰 선 부분과 같다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853909-db319267-facc-4aa3-b975-7cbd9db94fd5.png" alt="7"></p>
<p>연결하면 다음과 같은 화면이 뜨는데 vm으로 연결을 진행한다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853910-c37f7594-3f46-4e17-9c9b-03f05791498b.png" alt="8"></p>
<p>Nvidia 공홈에서 회원가입 후 로그인 진행</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853918-7b82df32-ada5-40be-958c-c9896080d73c.png" alt="13"></p>
<p>이때 리눅스 젯팩 버전을 4.4.1로 설정해줘야 한다.</p>
<p>Jetpack version이 안 뜰 경우, 터미널에서 <strong>sdkmanager --archivedversions</strong>로 검색을 해주면 해당하는 버전을 선택할 수 있고 이를 통해 바로 접속이 가능하다.</p>
<p>이후 젯슨나노는 아래와 같이 선택하여 주고, 위와 같이 3가지만 선택하여 준다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853919-d9f34231-74dc-4b4e-8322-0ebc0aedbc03.png" alt="14"></p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853921-47f60827-a140-4fbf-9bd9-ca73fcf18eb7.png" alt="18"></p>
<p>이후 다음과 같은 화면이 뜨는데 메뉴얼로 셋업을 변경하고 진행하면 된다.</p>
<p><img src="https://user-images.githubusercontent.com/84653623/219853925-74e9bd3f-191d-4f1c-bada-7879b5a29b11.png" alt="MicrosoftTeams-image"></p>
<p>설치가 다 되면 다음과 같은 화면이 뜬다!</p>
<p>설치가 완료되면 범퍼를 다시 빼주고, emmc로 부팅을 하여준다.</p>

<h2>3. emmc에서 usb로 부팅 바꿔주기</h2>
<p>다음의 유튜브 영상을 참고하여 그대로 진행한다.</p>
<p><a href="https://www.youtube.com/watch?v=xwH2qfsvGio">https://www.youtube.com/watch?v=xwH2qfsvGio</a></p>
<h3>jetson nano 내에서 입력하여 진행하면 된다.</h3>
<p>yahboom jetson nano는 emmc에 올라간 부트로더 extlinux가 /dev/emmc블라블라 ..</p>
<p>그러니까 emmc를 루트파일시스템으로 잡고 있었는데</p>
<p>이걸 usb에서 부팅할수 있도록 /boot/extlinux/extlinux.conf에서 루트파일시스템을 emmc가 아닌 /dev/sda1(위 영상의 경우)로 바꿔주면 된다고 한다.</p>
<p>문제는 2가지가 있었는데</p>
<ol>
<li>emmc에 깔린 jetpack 버전과 usb에 담긴 jetpack 버전이 같아야 한다.</li>
<li>꽂은 usb가 /dev/sda1로 읽히는지 확인</li>
</ol>
<p><strong>2번의 경우, df를 입력하여주면 확인할 수 있다.</strong></p>
<h3><strong>입력 순서</strong></h3>
<pre><code class="language-jsx">sudo apt-get update
sudo apt-cache show nvidia-jetpack
# 위에 코드로 버전 확인
cd /boot/extlinux/

sudo cp extlinux.conf extlinux.conf.boot_emmc_backup

ls
sudo vim extlinux.conf

</code></pre>
<p><img src="https://user-images.githubusercontent.com/84653623/219853924-b24752a4-6d23-4c70-9fba-dd3824d141f4.png" alt="20"></p>
<p>이후 sudo reboot 하면 os가 완벽하게 설치된다!</p>
<p>이후 id는 nano, pw는 yahboom으로 설정된다.</p>
