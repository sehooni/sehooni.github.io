<h1>1. minimal-mistakes-theme에서 back to top 버튼 만들기</h1>
<h2>1.1. <code>_sass/minimal-mistakes/_sidebar.scss</code>에서 아래 내용 삽입</h2>
<p>삽입할 내용은 다음과 같아요! 다음 내용의 수정을 통해서 위치 조정 또한 가능하답니다.</p>
<pre><code class="language-scss">.sidebar__top {
  position: fixed;
  bottom: 1.5em;
  right: 2em;
  z-index: 10;
}
</code></pre>
<p><strong>삽입 후</strong>의 모습은 다음과 같아요.</p>
<p><code>_sidebar.scss</code>를 보면 다음과 같이 ========= 부분 <strong>아래</strong>에 추가해주면 된다는 점.</p>
<pre><code class="language-scss">/* ==========================================================================
   SIDEBAR
   ========================================================================== */

/*
   Default
   ========================================================================== */

.sidebar__top {
  position: fixed;
  bottom: 1.5em;
  right: 2em;
  z-index: 10;
}

.sidebar {
  @include clearfix();
  // @include breakpoint(max-width $large) {
  //   /* fix z-index order of follow links */
  //   position: relative;
  //   z-index: 10;
  //   -webkit-transform: translate3d(0, 0, 0);
  //   transform: translate3d(0, 0, 0);
  // }
  --생략--
</code></pre>
