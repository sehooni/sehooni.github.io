<h1>3. Yolov5 설치</h1>
<h2>1. github에서 기본 repository 받아오기</h2>
<p>이제 95프로 왔다. 다온거다</p>
<p>아래의 코드를 통해 Yolov5의 repository를 받아준다.</p>
<pre><code class="language-bash">git clone https://github.com/ultralytics/yolov5
cd yolov5
 
# yolov5s.pt weight 다운로드
wget https://github.com/ultralytics/yolov5/releases/download/v6.0/yolov5s.pt
</code></pre>
<h2>2. 이후 폴더 내에 존재하는 requirements.txt에서 이미 설치한 항목들 지워주기</h2>
<p>2-5과정에서 이미 설치한 항목들이 있으므로 requirements.txt에서 제거해준다.</p>
<pre><code class="language-bash"># 다음 내용 requirements.txt에서 제거
 
numpy>=1.18.5
opencv-python>=4.1.2
torch>=1.7.0
torchvision>=0.8.1
</code></pre>
<h2>3. pip 업그레이드 이후 requirements.txt 설치</h2>
<pre><code class="language-bash">python3 -m pip install --upgrade pip
 
python3 -m pip install -r requirements.txt
</code></pre>
<p>이렇게 하면 yolov5 설치가 완료된다!</p>
<h1>4. 추가 참고 사항</h1>
<h2>FPS 2로 출력되는 문제 (1)</h2>
<p>다 설치하고 나면 FPS가 2로 출력되면서 웹캠으로 영상을 출력하는데 끊어지는 현상들이 발생한다.</p>
<p>본인의 판단으로는 아래의 사진에서 나타나는 경고 한 줄 때문이라 생각되는데..</p>

<h2>FPS 2로 출력되는 문제 (2) - [inf frames 2560x1472 at 2.00 FPS]</h2>
<p>inf frames 2560x1472 at 2.00 FPS 와 같이 출력되는 부분은 utils.dataloader.py에서 수정해 줄수 있는데 인위적으로 수정해서는 개선이 되지 않았다.</p>
<p>저 내용은 아래 사진들에서 볼 수 있듯이 환경의 종류가 다양한데, 이를 어떻게 적용하고 수정하는지 찾아봐야할 것 같다.(23.01.13)</p>
